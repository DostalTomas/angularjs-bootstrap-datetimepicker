DateObject

  1. - type in module bootstrap-datetimepicker

Properties

  • dateTime

    DateTime

    Luxon DateTime value of this date object.

  • selectable

    Boolean

    Indicates that date value is selectable by the user.

  • active

    Boolean

    Indicates that date object is part of the model value.

  • future

    Boolean

    Indicates that date value is after the date range of the current view.

  • past

    Boolean

    Indicates that date value is prior to the date range of the current view.

  • display

    String

    The way this value will be displayed on the calendar.
